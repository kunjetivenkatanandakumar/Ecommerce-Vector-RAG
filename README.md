# 🛒 AI Shopping Assistant using RAG

## Overview

An AI-powered shopping assistant that performs semantic product search using vector embeddings and generates intelligent recommendations using Google Gemini.

## Features

- Semantic Product Search
- Vector Database (ChromaDB)
- Sentence Transformer Embeddings
- Retrieval-Augmented Generation (RAG)
- Google Gemini Integration
- Streamlit Dashboard
- FastAPI Backend

## Tech Stack

- Python
- Pandas
- Scikit-learn
- Sentence Transformers
- ChromaDB
- Google Gemini
- FastAPI
- Streamlit
